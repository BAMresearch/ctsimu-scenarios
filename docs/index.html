CTSimU Scenario Descriptions 0.9

This is the specification for a JSON-based file format to handle the parameters of a full industrial CT scan scenario, intended for describing virtual CT scenarios for simulations, as well as for documenting real CT scan geometries and acquisition parameters and their measurement uncertainties.

Specification version 0.9, released under the Apache 2.0 license.
A PDF document of this specification is available.

About this work

This work was funded through the project CTSimU (WIPANO project 03TNH026A). WIPANO projects are financed by the German Federal Ministry for Economic Affairs and Energy and managed by Project Management Jülich.
